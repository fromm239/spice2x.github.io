about spice2x

spice2x is a fork of SpiceTools, focused on addressing bugs, usability, and quality-of-life improvements

spice2x binaries are meant to be drop-in replacements for existing installs (spice.exe, spice64.exe, spicecfg.exe).

We are not associated with the original SpiceTools developers. SpiceCompanion apps are also separate projects owned by the app developers.

Efforts will be made to keep up to date with upstream changes.

suggested addons

Check out these cool tools that can be used with spice:
  • SpiceCompanion Android and iOS - control spice from your phone / tablet
  • ifs_layeredfs - for live game asset modding; spicetools ships with one built-in, but it's an old version; recommend that you download a newer version instead and add it as a DLL hook.
  • REAL - audio latency reduction for shared mode WASAPI (works well with popn)
  • hypersonik - DirectSound audio latency reduction for IIDX 24 and below
  • popnhax - various mods to make popn more playable
  • FlexASIO - software ASIO stack with multiple backends
  • Asphyxia - offline EA emulator & plugins

issues, questions, bugs, feature requests

Please use the github repo below. Low effort asks for troubleshooting will always be closed without explanation. At the very least, please provide a log file with an explanation of what the issue is. Due to time availability and sensitivity of game data, not every question will go answered. Code submissions are welcome, but no guarantees that it'll be merged.
