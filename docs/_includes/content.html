about spice2x

spice2x is a fork of SpiceTools, focused on addressing bugs, usability, and quality-of-life improvements

spice2x binaries are meant to be drop-in replacements for existing installs (spice.exe, spice64.exe, spicecfg.exe).

We are not associated with the original SpiceTools developers. SpiceCompanion apps are also separate projects owned by the app developers.

Efforts will be made to keep up to date with upstream changes.

known issues

Some anti-virus software may (incorrectly) flag the binaries as malware. If you have concerns, you are free to inspect the included source files and compile on your own.

Gitadora boots but hangs trying to display card-in scene on some AMD Ryzen processors - e.g., 3900X, 5950X, and others. This is a known hardware issue. As a workaround, try disabling SMT in the BIOS, or try Process Affinity option in spice2x and limit the game to half the processors available.

EG may launch at wrong refresh rate when using the auto-rotate option. Please enable -9on12 and -graphics-force-refresh to work around this.

When in windowed mode, resizing the window also resizes the overlay, which may cause it to be difficult to use. Additionally, there is also no linear filtering that can be enabled in windowed mode. It's unlikely that these issues will be solved in the future.

Multi-monitor games (TDJ, VM, etc) don't play nice with -monitor option. As a workaround, change the primary monitor before launching.

Multi-monitor games (TDJ, VM, etc) don't work well with windowed mode (-w). Namely, interacting with subscreen will not work as expected. Please use full screen mode.

For more, please see the issue tracker.

applying patches (hex edits)

Patches are not being updated. Please use Bemani Patcher or other similar sites.

If you want to have patches show up in Patch Manager in the overlay, please follow these instructions:

  1. Create patches.json. For an example, download this file: patches.json
  2. Modify the file, following the format. Note that dataOffset field is in decimal, not hex.
  3. Place the file next to Spice executables. The game will load it automatically.

suggested addons

Check out these cool tools that can be used with spice:
  • SpiceCompanion Android and iOS - control spice from your phone / tablet
  • ifs_layeredfs - for live game asset modding; spicetools ships with one built-in, but it's an old version; recommend that you download a newer version instead and add it as a DLL hook.
  • REAL - audio latency reduction for shared mode WASAPI (works well with popn)
  • hypersonik - DirectSound audio latency reduction for IIDX 24 and below
  • popnhax - various mods to make popn more playable
  • FlexASIO - software ASIO stack with multiple backends
  • Asphyxia - offline EA emulator & plugins

issues, questions, bugs, feature requests

Please use the github repo below. Low effort asks for troubleshooting will always be closed without explanation. At the very least, please provide a log file with an explanation of what the issue is. Due to time availability and sensitivity of game data, not every question will go answered. Code submissions are welcome, but no guarantees that it'll be merged.
